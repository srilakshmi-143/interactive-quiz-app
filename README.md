# 🧠 Interactive Quiz Web Application

## 📌 Project Overview

The **Interactive Quiz Web Application** is a browser-based quiz developed using **HTML**, **CSS**, and **JavaScript**. It allows users to answer different types of questions, including multiple-choice, checkbox-based, and fill-in-the-blank questions, and instantly displays their score with performance feedback.

---

## 🚀 Features

* ✅ Multiple Choice Questions (MCQs)
* ☑️ Multiple Select Questions
* ✍️ Fill in the Blank Questions
* 📊 Automatic Score Calculation
* 🎉 Instant Result Display
* 💬 Performance Feedback
* 📱 Responsive User Interface

---
